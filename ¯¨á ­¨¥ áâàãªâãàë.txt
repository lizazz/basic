В данном файле - находится описание папок не входящих в BASIC версию Yii 2.0

components
 |_ extansions - папка для размещения сторонних классов, созданных иными разработчиками, не имеющих отношение к текущему проекту (например API платежной системы);
 |_ helpers - классы пользовательских быстрых помощников, по типу преобразования чисел или обработки текста, и другие прочие часто используемые мелкие операции;
 |_ lib - классы - наподобие моделей,  но в пользовательском стиле, так же могуб быть класы обьектно процедурной структуры;

widgets - папка для создания пользовательских виджетовю. Виджет создается по следующей структуре:
 |_WidgetName     - папка виджета с именем создаваемого виджета;
   |_WidgetName.php - Входной класс с именем создаваемого виджета;
   |_WidgetNameModel.php - Необязательный Класс Модель с именем создаваемого виджета и суффиксом Model для разного рода вычислительных действий, касаемых данного виджета;
   |_views - папка представлений виджета (необязательно);
     |_first.php -  представление с произвольным именем (необязательно);
     |_secondary.php - 	представление с произвольным именем (необязательно);	

www - переименованная стандартная папка web, содержимое которой доступно пользователю;

www/css - стили проекта;

www/scripts - скрипты проекта;

www/images - картинки проекта;

www/(css | images | scripts)
 |_admin - ресурсы суперадминистраторской части сайта;
 |_common - общие ресурсы сайта;
 |_frontend - ресурсы пользовательской части сайта;
 |_vendor - ресурсы адимистраторской сайта;
